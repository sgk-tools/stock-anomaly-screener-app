株式アノマリー候補抽出 v1.1

■目的
日本株・米国株・ETFを対象に、過去アノマリー、簡易CAN SLIM、トレンド、出来高から「確認する価値がある候補」を抽出する。

■重要
これは売買推奨アプリではありません。
「買い候補」「注目ETF」は、即買いではなく個別確認に進む候補です。
決算、EPS成長、売上成長、財務、地合い、チャート形状、損切り位置は別途確認してください。

■iPhoneでの使い方
1. GitHub PagesのURLをSafariで開く。
2. 「最新データ取得」を押す。
3. 総合、日本株、米国株、ETF、見方のタブを切り替える。
4. 問題なければSafari共有ボタンから「ホーム画面に追加」。

■表示項目
・総合スコア
・アノマリースコア
・簡易CAN SLIMスコア
・トレンドスコア
・同月勝率
・同月平均騰落率
・3か月/6か月騰落率
・52週高値差
・出来高倍率
・市場地合い

■簡易CAN SLIMについて
Stooqの日足データだけで見られる以下を簡易評価します。
N：52週高値接近
S：出来高倍率
L：3か月・6か月・12か月モメンタム
M：指数ETFの市場地合い

C：四半期EPS成長
A：年間EPS成長
I：機関投資家保有
これらは日足データだけでは確認できないため、要個別確認です。

■ETFについて
ETFは企業ではないため、CAN SLIM完全判定には向きません。
ETFタブでは、アノマリー、モメンタム、移動平均、52週高値接近、出来高で確認します。

■データについて
株価・出来高・過去日足はStooq一括ヒストリカルデータを想定しています。
Stooq側の認証・配布仕様により、GitHub Actionsからの自動取得が失敗する場合があります。
その場合は、手動で data/raw/ に以下を配置してください。
・d_us_txt.zip
・d_jp_txt.zip
